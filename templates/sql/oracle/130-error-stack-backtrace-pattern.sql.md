###### Oracle PL/SQL
### Re-raise with context, keep stack
Never swallow the stack; attach context then re-raise.
```sql
BEGIN
  -- some code
  RAISE_APPLICATION_ERROR(-20001, 'oops');
EXCEPTION
  WHEN OTHERS THEN
    log_err('my_module',
      DBMS_UTILITY.format_error_stack || CHR(10) ||
      DBMS_UTILITY.format_error_backtrace);
    RAISE;
END;
/
```
